# macOS 标签编辑与键盘交互设计

## 目标

改善 macOS Shop! 在物品编辑、侧栏标签管理与键盘操作上的体验，与 iOS 编辑器能力对齐，同时保留 mac 三栏布局习惯。

## 已确认决策

| 主题 | 决策 |
|------|------|
| 保存模式 | **B**：新建草稿显式保存；编辑已有物品即时保存 |
| 侧栏标签管理 | **A**：右键菜单（重命名 / 改颜色 / 删除 + 新建标签） |
| 编辑 Esc | 取消选中，回到占位详情（即时保存无法「放弃未保存」） |
| 新建 Esc | 名称非空则保存，否则放弃草稿 |
| 编辑 ⌘Z | 撤回最近一次物品字段修改（名/标签/日期） |
| 侧栏标签匹配 | 任一 / 全部（`tagMatchMode`） |
| 日期字段 Tab / ↑↓ | 离开日期，切换编辑器字段（不改数值） |
| 日期字段 ←→ | 年/月/日/时/分段 |

## 编辑区：标签与新建

- 始终显示「标签」区块（无标签时也显示）
- 已有标签：可聚焦 Button 行 + 勾选态（替代 checkbox Toggle）
- 列表下方内联新建：名称 + 色板 +「添加标签」，对齐 iOS / 设置页
- 新建成功后自动选中新标签

## 键盘

| 场景 | 行为 |
|------|------|
| Tab | 名字 → 各标签行 → 新建标签名 → 添加时间 |
| 标签行获焦 + Space | 切换该标签选中 |
| 新建草稿 | ⌘S / Enter（名非空）保存；Esc：名非空保存，否则放弃 |
| 编辑已有 | 即时保存；⌘S / Enter 显式 save；Esc 取消选中 |
| 详情区有焦点 | 不触发列表 Space/Return 完成快捷键 |
| 日期字段 | Tab / ↑↓ 切换编辑字段；←→ 换日期段 |

## 侧栏标签右键

每个标签：重命名… / 更改颜色（子菜单）/ 删除（带 undo）

「全部标签」行与标签区空白：新建标签…

设置页 `MacTagSettingsTab` 保留。

## 实现要点

- 共享组件 `MacTagEditorComponents.swift`
- `DataStore.updateItem` 增加 `presentUndo`；`ShoppingUndo.undoItemEdit`
- 扩展 `isDetailTextFocused` 覆盖标签与日期焦点
