# iPhone 与 Mac 多选设计

## 目标

为 iPhone 和 Mac 的购物列表加入一致的多选与批量操作能力。Mac 同时支持普通点击单选、Command 点击逐项切换和 Shift 点击连续范围选择；iPhone 支持工具栏进入多选及长按条目快速进入。

## 支持的批量操作

- 将全部所选条目标记为完成
- 将全部所选条目恢复为待买
- 批量添加或移除标签
- 批量删除（始终二次确认）

完成和恢复是两个独立操作。混合选择待买与已完成条目时，“全部完成”仅确保所有条目最终为完成，“全部恢复”仅确保所有条目最终为待买，不对每个条目简单取反。

## 共享状态与数据层

两端使用 `Set<UUID>` 保存选择，以 ID 而不是模型引用作为稳定身份。选择集合由各平台视图持有，不参与持久化或同步。

`DataStore` / `ShoppingStore` 增加批量 mutation API：

- 批量设置统一完成状态
- 批量软删除
- 批量添加标签
- 批量移除标签

每个批量操作在一次 mutation 中完成，只触发一次数据刷新、Widget 发布与同步调度。完成、恢复、删除和标签修改都生成一个整体撤销动作；撤销恢复所有受影响条目的原始状态。

当筛选、搜索、同步或删除导致已选 ID 不再存在或不可见时，视图将其从选择集合移除。选择集合变空时自动退出多选模式。

## iPhone 交互

- 导航栏右上角新增“选择”入口。
- 长按任意条目进入多选并选中该条目。
- 多选模式下，点击整行切换选中状态；原有编辑、完成按钮和滑动操作暂时禁用，避免动作歧义。
- 每行左侧显示选择圆圈，使用品牌色表达选中。
- 底部批量操作栏显示“已选 N 项”，并提供完成、恢复、标签和删除。
- “完成”和“恢复”始终是独立动作；对已经处于目标状态的条目不做无意义更新。
- 标签按钮弹出三态标签面板。
- 删除按钮弹出包含所选数量的确认对话框。
- “完成”或“恢复”后，仍可继续选择剩余可见条目；删除成功后清空选择并退出。

## Mac 交互

- 普通点击：清除旧选择，仅选择当前条目，并在右侧显示单项详情。
- Command 点击：逐项加入或移出当前选择。
- Shift 点击：以最近一次锚点和当前条目为端点，按当前列表的视觉顺序选择连续范围；范围可跨分组和待买/归档 Section。
- Shift + Command 点击：在保留现有选择的基础上加入连续范围。
- 草稿行不参与多选；开始新增草稿时清空多选。
- 单选时保留现有详情编辑器；选中多项时右侧切换为批量操作面板。
- 批量面板顶部显示“已选 N 项”，下方提供全部完成、全部恢复、标签和删除。
- Space / Return 在单选时保持原行为；多选时不做状态取反，避免混合状态歧义，批量状态必须通过明确的“全部完成”或“全部恢复”操作。
- 自定义品牌色行背景继续作为选择反馈，不启用系统灰蓝选择高亮。

## 标签三态

对每个标签计算：

- `all`：所有所选条目都拥有该标签
- `some`：仅部分所选条目拥有该标签
- `none`：没有所选条目拥有该标签

视觉上分别使用勾选、短横线和空圆圈。点击 `all` 时从全部所选条目移除；点击 `some` 或 `none` 时为全部所选条目添加。

## 触感、动画与可访问性

- 进入多选、改变单项选择及执行批量操作时提供平台适合的单次触感反馈。
- 批量操作按钮包含可访问性标签和已选数量。
- 选择行应暴露“已选择”状态；三态标签应读出“全部拥有 / 部分拥有 / 未添加”。
- 遵循 Reduce Motion，避免强制选择动画。

## 错误与一致性

- 批量 mutation 失败时不部分提交，不生成撤销动作，并沿用现有 DataStore 错误展示。
- 删除使用软删除语义，后续照常参与 WebDAV tombstone 同步。
- 批量完成为每个发生变化的条目设置一致操作时间；已完成条目保持原 `completedAt`。
- 批量恢复清空受影响条目的 `completedAt`。
- 标签操作只改变标签关系，不改变名称、时间或完成状态。

## 测试

共享层：

- 混合状态选择执行全部完成与全部恢复
- 批量删除及整体撤销
- 三态标签的添加、移除与整体撤销
- 操作包含不存在 ID 时安全忽略
- 一次批量操作仅产生一次 mutation 通知

iPhone：

- 工具栏与长按均可进入多选
- 多选时行点击不打开编辑器
- 删除确认显示数量
- 批量操作后选择状态正确

Mac：

- 普通、Command、Shift、Shift + Command 选择语义
- 连续范围遵循筛选、排序和分组后的视觉顺序
- 多选详情面板与已选数量
- 草稿、同步和筛选变化时选择清理
- 多选时 Space / Return 不进行逐项取反
